For General Purpose#

For Document#

  • Microsoft’s Copilot
  • Notion AI
  • ChatDoc/ChatPDF, upload PDF and analyze.

For Code#

  • Github’s Copilot X, exsiting Copilot costs $10/mo after 60d trial.
    • Using Copilot for Docs to learn a SDK/framework/API.
  • Cursor, or CodeCursor(vscode plugin), analyze opened document/code, currently FREE.
  • Open AI’s CodeX

For 3D/2D Art#

  • Stable-Diffusion(Github), an open-sourced model can run on PC.
    • Using ControlNet (Github )to add more control to this model.
    • Find/Share self-tuned models on civitai
  • Midjourney, tuned from stable diffusion, famous for its artistic style, 25 times FREE try.
  • Open AI’s DALL-E-2
  • Adobe’s Firefly
  • Bing’s Image Creator
